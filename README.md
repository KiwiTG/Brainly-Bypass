# Brainly-Bypass-Windows

FIREFOX IS REQUIRED!!!

[*] What does it do?

  - It works by clearing the cookies of Brainly each visit so you dont have to watch an ad to see the answer to a question.

Install:

   Install the Requirements

    pip install -r requirements.txt

    python bypass.py
